# Course Labs

This directory contains the numbered lab sequence for the ClosedLoop.AI Claude Code Expert Training course.

## Lab Sequence

1. `00-no-speed-limits-setup-local-stack-bootstrap` — No Speed Limits Setup & Local Stack Bootstrap
2. `01-know-the-machine-claude-code-primitives` — Know the Machine: Claude Code Primitives
3. `02-from-mess-to-mission-creating-plans-for-execution` — From Mess to Mission: Creating Plans for Execution.
4. `03-commit-archaeology-find-the-why-before-the-what` — Commit Archaeology: Find the Why Before the What
5. `04-give-claude-eyes-and-ears-connecting-dynamic-data-sources` — Give Claude Eyes & Ears: Connecting Dynamic Data Sources
6. `05-review-test-and-verify` — Review, Test, & Verify
7. `06-agent-orchestra-design-a-multi-agent-delivery-workflow` — Agent Orchestra: Design a Multi-Agent Delivery Workflow
8. `07-build-your-agentic-operating-system` — Build Your Agentic Operating System

## How to Use the Labs

The labs are designed to be completed in order. Each lab establishes concepts, artifacts, or operating constraints used by the labs that follow.

- Lab 00 establishes the working environment and baseline rules.
- Labs 01 through 05 build the primitives, planning, investigation, dynamic context, review, and verification stack.
- Lab 06 is the capstone orchestration lab that explicitly reuses artifacts from Labs 00 through 05.
- Lab 07 is the operating-system capstone that makes the Lab 06 workflow observable, reviewable, and self-improving.
