# Course Labs

This directory contains the numbered lab sequence for the ClosedLoop.AI Claude Code Expert Training course.

## Lab Sequence

1. `00-setup-onboarding` — No Speed Limits Setup & Local Stack Bootstrap
2. `01-primitives` — Know the Machine: Claude Code Primitives
3. `02-planning` — From Mess to Mission: Creating Plans for Execution.
4. `03-history-issues` — Commit Archaeology: Find the Why Before the What
5. `04-feedback-tools` — Give Claude Eyes & Ears: Connecting Dynamic Data Sources
6. `05-review-testing` — Diff Court: Review, Test, Prove
7. `06-workflow-integration` — Agent Orchestra: Design a Multi-Agent Delivery Workflow
8. `07-operating` — Build Your Agentic Operating System

## How to Use the Labs

The labs are designed to be completed in order. Each lab establishes concepts, artifacts, or operating constraints used by the labs that follow.

- Lab 00 establishes the working environment and baseline rules.
- Labs 01 through 05 build the primitives, planning, investigation, dynamic context, review, and verification stack.
- Lab 06 is the capstone orchestration lab that explicitly reuses artifacts from Labs 00 through 05.
- Lab 07 is the operating-system capstone that makes the Lab 06 workflow observable, reviewable, and self-improving.

Legacy `day*` folders remain in the repository as earlier draft material, but the numbered sequence is the authoritative course path.
