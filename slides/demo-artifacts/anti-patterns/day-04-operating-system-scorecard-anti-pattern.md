# Day 4 Anti-Pattern Artifact

## Workflow Metrics

The workflow is improving.

- quality seems better
- people are happier
- it feels faster
- the agents are useful

## Changes For Next Time

- keep refining the workflow
- maybe improve prompts
- maybe change the model

## Why This Is Weak

- no actual metrics
- no telemetry points
- no baseline or current value
- no link between observations and actions
- no review categories
- no next-run checklist
- “feels faster” is not operational evidence
