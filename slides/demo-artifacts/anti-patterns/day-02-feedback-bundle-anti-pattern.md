# Day 2 Anti-Pattern Artifact

## Feedback Bundle

The last fix did not work.

Please look again and fix it properly.

I think the discount logic is still broken somewhere in the function.

Maybe check the tests and whatever else seems relevant.

## Why This Is Weak

- no failing test name
- no CLI command
- no observed versus expected behavior
- no constraint on what should and should not change
- no dynamic artifact at all
- the downstream model has to guess what failed
